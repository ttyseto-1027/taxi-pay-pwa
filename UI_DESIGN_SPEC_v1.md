# 画面設計書 Version 1.1

## 1. 初回設定画面
- 勤務区分選択（12種類）
- 勤務条件表示
- 免責事項確認

## 2. メイン画面
- 右上：⚙️設定
- 給与月切替
- 給与サマリー
- 給与明細形式内訳
- 印刷／PDF、CSV、給与締め

## 3. 勤務入力画面
- 勤務日
- 税込営収
- 税抜営収（表示専用）
- 出勤時刻（アルコール）
- 退勤時刻（アルコール）
- 通常休憩（時間・分）
- 深夜休憩（時間・分）
- 休日区分
- 事故あり、違反あり

## 4. 設定画面
- 勤務区分
- 社会保険料等、住民税、組合費、共済費
- 扶養親族等の数
- 源泉徴収区分（甲・乙）
- 有休日数、標準報酬日額
- 消費税率・所定時間（表示専用）

## 5. 開発者設定
係数、手当額、割増率を設定する。通常利用者向け導線からはパスワード入力後に開く。

## メイン画面のセクション順
1. 勤務実績を入力
2. 日別明細
3. yyyy年mm月 給与シミュレーション
4. 給与締め履歴
5. 免責事項・バージョン表記
