# 変更履歴

## Version 1.0β レイアウト改善（2026-07-13）
- メイン画面の表示順を、利用頻度に合わせて次の順番へ変更。
  1. 勤務実績を入力
  2. 日別明細
  3. yyyy年mm月 給与シミュレーション
  4. 給与締め履歴
- 免責事項・バージョン表記を画面最下部へ移動。
- Service Workerのキャッシュ名を更新。

## Version 0.9.0 beta（2026-07-13）
- 勤務区分12種類を初回選択・端末保存する仕様を維持。
- 日報入力に「出勤時刻（アルコール）」「退勤時刻（アルコール）」を追加。
- 通常休憩と深夜休憩を「時間・分」に分けて入力できるよう変更。
- 休日区分（通常、法定休日、法定外休日）を追加。
- 実働時間、深夜労働時間、所定時間外、法定時間外、月60時間超、休日労働の概算計算を追加。
- 歩合時間単価を基礎とした割増賃金の概算計算を追加。
- 国税庁「令和8年分 給与所得の源泉徴収税額表（月額表）」を組み込み、甲欄・乙欄、扶養親族等の数に応じた所得税の自動計算を追加。
- 積算歩合給の表示名を勤務区分別に変更。
  - 隔日歩合給A・B
  - 昼日勤歩合給A・B・C
  - 夜日勤歩合給A・B
  - 定隔・定昼・定夜積算歩合給
- 給与明細と照合しやすい内訳表示を追加。
- Service Workerのキャッシュ名を更新。

## Version 0.8.0 beta
- 通常勤務3種類・定時制9種類に対応。
- 勤務区分別の積算歩合給計算を追加。

## Version 1.0β - ブランド・ベータ運用表示更新
- 個人名を表示しない運用方針へ統一
- ベータ版では問い合わせ先を設置しない方針を明記
- アプリ最下部の注意書き・バージョン・権利表示を指定文面へ統一
- 設定画面にベータ版利用条件とデータ取扱いの概要を追加
- ベータ版利用条件、プライバシーポリシー、ブランドガイドラインを追加
- Service Workerのキャッシュ名を更新

## Version 1.0β - Mobile layout fix
- PC・スマートフォン共通で、表示順を「勤務実績を入力」→「日別明細」→「給与シミュレーション」→「給与締め履歴」に固定。
- Service Workerのキャッシュ名を更新。

## Version 1.0β 更新：休憩時間入力
- 通常休憩・深夜休憩を切り替えず、各項目を常時表示する方式を維持。
- 時間と分をそれぞれ「−／＋」ボタンで1単位ずつ増減する入力方式へ変更。
- 時間は0〜24、分は0〜59の範囲で設定可能。
- 15分刻みなどの固定ステップは使用せず、実際の休憩時間を入力可能。
