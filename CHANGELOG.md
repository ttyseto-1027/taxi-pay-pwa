# CHANGELOG

## Version 1.1β
- 管理画面に既存無料コードの利用上限変更機能を追加
- 同一コード再発行による usageCount リセットを防止
- 日別入力に「通常勤務・1有給・2有給」を追加
- 有給付与日、付与日数、残日数、付与・使用履歴を追加
- 隔日勤務の1有給を7時間30分、2有給を14時間15分・1乗務相当として処理
- 有給のみなし時間を割増賃金計算から除外
- CSV・給与締め履歴へ有給情報を追加


## 1.0β 更新
- 休憩時間入力を「−／＋」連打方式から、時間（0〜24）・分（0〜59）の選択式へ変更。
- スマートフォンではOS標準の選択ピッカーが開くため、連続タップによる画面拡大を防止。
# 変更履歴

## Version 0.9.0 beta（2026-07-13）
- 勤務区分12種類を初回選択・端末保存する仕様を維持。
- 日報入力に「出勤時刻（アルコール）」「退勤時刻（アルコール）」を追加。
- 通常休憩と深夜休憩を「時間・分」に分けて入力できるよう変更。
- 休日区分（通常、法定休日、法定外休日）を追加。
- 実働時間、深夜労働時間、所定時間外、法定時間外、月60時間超、休日労働の概算計算を追加。
- 歩合時間単価を基礎とした割増賃金の概算計算を追加。
- 国税庁「令和8年分 給与所得の源泉徴収税額表（月額表）」を組み込み、甲欄・乙欄、扶養親族等の数に応じた所得税の自動計算を追加。
- 積算歩合給の表示名を勤務区分別に変更。
  - 隔日歩合給A・B
  - 昼日勤歩合給A・B・C
  - 夜日勤歩合給A・B
  - 定隔・定昼・定夜積算歩合給
- 給与明細と照合しやすい内訳表示を追加。
- Service Workerのキャッシュ名を更新。

## Version 0.8.0 beta
- 通常勤務3種類・定時制9種類に対応。
- 勤務区分別の積算歩合給計算を追加。

## Version 1.0β - ブランド・ベータ運用表示更新
- 個人名を表示しない運用方針へ統一
- ベータ版では問い合わせ先を設置しない方針を明記
- アプリ最下部の注意書き・バージョン・権利表示を指定文面へ統一
- 設定画面にベータ版利用条件とデータ取扱いの概要を追加
- ベータ版利用条件、プライバシーポリシー、ブランドガイドラインを追加
- Service Workerのキャッシュ名を更新

## Version 1.0β 更新（2026-07-13）
- スマートフォンで休憩時間の増減ボタンが反応しない問題を修正。
- 通常休憩・深夜休憩の時間と分を、PC版と同じ「−／＋」ボタンで1単位ずつ増減する表示に統一。
- 時間は0〜24、分は0〜59の範囲に制限。
- スマートフォンで押しやすい48px以上のタップ領域とタッチ操作設定を追加。
- PC版とスマートフォン版の画面順を同一に固定。

## Version 1.0β - Firebase認証対応

- 氏名、メールアドレス、本人専用パスワードによる新規登録を追加
- β版無料利用コードを追加
- Firebase AuthenticationとCloud Firestoreへ対応
- 管理者専用ユーザー一覧、人数集計、利用停止・利用再開を追加
- 給与・営収・勤務実績は端末内保存を維持
- 模範勤務手当を、所定労働時間を満たした乗務が所定乗務数へ達した場合のみ支給するよう修正
