# タクシー給与シミュレーター PWA版

タクシー乗務員が各自のスマートフォンで、給与の概算を確認するためのPWAです。

本アプリは会社の正式な給与計算システムを代替するものではありません。乗務員個人が日々の営業実績をもとに、概算の総支給額・手取り額を確認するための補助ツールです。

## 使い方

1. `index.html` をブラウザで開く
2. スマートフォンでは公開URLをSafariまたはChromeで開く
3. ホーム画面に追加して利用する

## 主な機能

- 日報入力
- 税込営収のみ入力
- 税抜営収の自動計算
- 給与算定営業収入の自動計算
- 歩合給A/Bの自動計算
- 諸手当の自動計算
- 通常休憩・深夜休憩入力
- 概算総支給・概算手取り表示
- 給与締め
- 印刷 / PDF保存
- CSV出力
- 管理者設定

## 同梱ドキュメント

- `PROJECT_PROPOSAL_UNION_v1.md`：企画書（組合提出用）
- `SPEC_v1.md`：簡易仕様書
- `SYSTEM_SPEC_v1.md`：システム仕様書
- `UI_DESIGN_SPEC_v1.md`：画面設計書
- `PAYROLL_CALC_SPEC_v1.md`：給与計算仕様書
- `DATABASE_DESIGN_v1.md`：データベース設計書
- `MANUAL_user.md`：利用者マニュアル
- `MANUAL_admin.md`：管理者マニュアル

## データ保存

利用者の入力データは端末内に保存されます。

- サーバー送信なし
- ログイン不要
- 管理者による利用者データ閲覧なし

## 注意事項

本アプリで表示される給与額は概算です。正式な給与額は会社が発行する給与明細を確認してください。
