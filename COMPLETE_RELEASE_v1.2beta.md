# v1.2β 全ファイル完成版

このリリースは、受領したGitHubリポジトリ全体を基礎に作成しています。

## 主な整合化

- 利用者・管理者をGoogleログインへ統一
- Googleメール許可リスト
- 初回招待コード登録
- 招待コードの利用上限（初期値10人）
- 利用者の停止・再開
- 管理者権限をFirestoreの `admins/{uid}` で判定
- Firestore Security Rulesを認証方式に合わせて更新
- Service Workerのキャッシュ番号を更新
- Googleログインでは使用しない旧パスワード変更欄を画面から削除
- データ取扱い説明を現在の実装に合わせて修正

給与計算ロジック、税額表、既存仕様書類は受領したリポジトリの内容を保持しています。


## 2026-07-21 管理者利用者登録バイパス

有効な `admins/{UID}` を持つ管理者は、許可リスト・招待コード・`users` 登録なしで利用者画面を利用できるように変更しました。一般利用者の認証条件は変更していません。
