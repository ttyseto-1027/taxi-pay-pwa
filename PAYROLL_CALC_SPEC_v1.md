# タクシー給与シミュレーター
## 給与計算仕様書 Version 1.0

---

## 1. 本書の位置づけ

本書は、タクシー乗務員が自身の営業実績を入力し、給与の概算を確認するための「タクシー給与シミュレーター」における給与計算仕様を定義する。

本アプリは、会社の正式な給与計算システムを代替するものではない。あくまで乗務員個人が、月途中または給与締め時点で、概算の総支給額・控除額・手取り額を把握するための補助ツールである。

正式な給与額は、会社が発行する給与明細および会社の給与計算規程に従う。

---

## 2. 計算の基本方針

### 2.1 入力を簡素化する方針

利用者の日々の入力項目は、可能な限り少なくする。

日報入力では、利用者は税込営収のみを入力し、税抜営収や給与算定営業収入は自動計算とする。

### 2.2 管理者設定を隠す方針

以下のような係数・率・手当額は、通常利用者には表示しない。

- 運賃改定係数
- 給与算定係数
- 歩合給A率
- 歩合給B率
- B開始額
- B最高算定額
- 模範勤務手当
- 無事故手当
- 無違反手当
- 各種割増率

これらは管理者モードでのみ確認・変更可能とする。

### 2.3 概算計算であること

社会保険料、厚生年金、所得税、住民税などは自動算出せず、利用者が給与明細等を参考に概算額を入力する。

## 3. 給与対象期間

### 3.1 賃金締め日

賃金締め日は以下とする。

| 月 | 締め日 |
|---|---:|
| 1月 | 15日 |
| 2月 | 14日 |
| 3月 | 16日 |
| 4月 | 15日 |
| 5月 | 15日 |
| 6月 | 15日 |
| 7月 | 15日 |
| 8月 | 15日 |
| 9月 | 15日 |
| 10月 | 15日 |
| 11月 | 15日 |
| 12月 | 15日 |

### 3.2 給与対象月の判定

勤務日が締め日以前の場合、その勤務は当月給与に含める。

勤務日が締め日の翌日以降の場合、その勤務は翌月給与に含める。

例：

| 勤務日 | 対象給与 |
|---|---|
| 2026年7月15日 | 2026年7月給与 |
| 2026年7月16日 | 2026年8月給与 |

### 3.3 給与対象期間の例

7月給与の対象期間は、原則として以下とする。

```text
6月16日 〜 7月15日
```

ただし、2月・3月は締め日の例外により対象期間が変動する。

## 6. 営収計算

### 6.1 税抜営収

税込営収から税抜営収を自動算出する。

```text
税抜営収 = 税込営収 ÷ (1 + 消費税率)
```

例：

```text
税込営収 110,000円、消費税率10%の場合
税抜営収 = 110,000 ÷ 1.10 = 100,000円
```

端数処理は、管理者設定で定義する。初期仕様では円未満四捨五入とする。

### 6.2 月間税抜営収

給与対象期間内の日報の税抜営収を合算する。

```text
月間税抜営収 = 対象期間内の税抜営収合計
```

### 6.3 給与算定営業収入

労働条件通知書の記載に基づき、月間税抜営収に運賃改定係数および給与算定係数を乗じる。

```text
給与算定営業収入 = 月間税抜営収 × 運賃改定係数 × 給与算定係数
```

初期値：

| 項目 | 値 |
|---|---:|
| 運賃改定係数 | 1 |
| 給与算定係数 | 0.9585 |

---

## 7. 歩合給計算

### 7.1 歩合給A

```text
歩合給A = 給与算定営業収入 × 歩合給A率
```

初期値：

```text
歩合給A率 = 41.44%
```

### 7.2 歩合給B

給与算定営業収入がB開始額以上の場合のみ、歩合給Bを支給する。

B最高算定額を上限として算定する。

```text
B対象額 = MIN(給与算定営業収入, B最高算定額) - B開始額
```

```text
歩合給B = MAX(B対象額, 0) × 歩合給B率
```

初期値：

| 項目 | 値 |
|---|---:|
| B開始額 | 420,000円 |
| B最高算定額 | 1,200,000円 |
| 歩合給B率 | 19.05% |

### 7.3 基本賃金

労働条件通知書の記載に従い、基本賃金は積算歩合給により支給されるものとする。

本アプリでは、以下を基本賃金相当として扱う。

```text
基本賃金相当 = 歩合給A + 歩合給B
```

## 8. 手当計算

### 8.1 模範勤務手当

月額固定の手当として加算する。

```text
模範勤務手当 = 管理者設定値
```

初期値：

```text
3,000円/月
```

### 8.2 無事故手当

事故なしの乗務に対して、1乗務ごとに加算する。

```text
無事故手当 = 事故なし乗務数 × 無事故手当単価
```

初期値：

```text
700円/乗務
```

### 8.3 無違反手当

違反なしの乗務に対して、1乗務ごとに加算する。

```text
無違反手当 = 違反なし乗務数 × 無違反手当単価
```

初期値：

```text
200円/乗務
```

### 8.4 手当合計

```text
手当合計 = 模範勤務手当 + 無事故手当 + 無違反手当
```

## 9. 労働時間計算

### 9.1 拘束時間

出庫時刻から帰庫時刻までを拘束時間とする。

帰庫時刻が出庫時刻より早い場合は、翌日帰庫として計算する。

例：

```text
出庫 15:00、帰庫 11:45 の場合
拘束時間 = 20時間45分
```

### 9.2 休憩時間

休憩時間は通常休憩と深夜休憩に分けて入力する。

```text
総休憩時間 = 通常休憩 + 深夜休憩
```

### 9.3 実働時間

```text
実働時間 = 拘束時間 - 通常休憩 - 深夜休憩
```

### 9.4 深夜労働時間

深夜時間帯は22:00〜翌5:00とする。

深夜労働時間は、勤務時間が深夜時間帯に重なる時間から深夜休憩を差し引いて算出する。

```text
深夜労働時間 = 深夜時間帯内の勤務時間 - 深夜休憩
```

深夜労働時間が0未満になる場合は0とする。

### 9.5 残業時間

1乗務ごとの残業時間は、実働時間から勤務区分別の1乗務所定労働時間を差し引いて算出する。

```text
残業時間 = MAX(実働時間 - 1乗務所定労働時間, 0)
```

月間残業時間は、給与対象期間内の残業時間を合算する。

## 10. 割増賃金計算

### 10.1 割増率

初期値は以下とする。

| 区分 | 割増率 |
|---|---:|
| 所定超時間外 | 25% |
| 法定超時間外 | 25% |
| 月60時間超 | 50% |
| 法定休日 | 35% |
| 法定外休日 | 25% |
| 深夜労働 | 25% |

### 10.2 割増単価

労働条件通知書上、基本賃金が積算歩合給であるため、割増単価の厳密な算出方法は会社規程に依存する。

本アプリでは概算計算として、以下の方法を採用する。

```text
概算時間単価 = 基本賃金相当 ÷ 月間所定労働時間
```

```text
割増賃金 = 概算時間単価 × 対象時間 × 割増率
```

### 10.3 深夜割増

```text
深夜割増 = 概算時間単価 × 月間深夜労働時間 × 深夜割増率
```

### 10.4 時間外割増

```text
時間外割増 = 概算時間単価 × 月間残業時間 × 時間外割増率
```

月間残業時間が60時間を超える場合、60時間を超えた部分には月60時間超割増率を適用する。

```text
60時間以内時間外 = MIN(月間残業時間, 60時間)
60時間超時間外 = MAX(月間残業時間 - 60時間, 0)
```

```text
時間外割増 =
  概算時間単価 × 60時間以内時間外 × 25%
+ 概算時間単価 × 60時間超時間外 × 50%
```

### 10.5 休日割増

日報の休日区分に応じて、法定休日または法定外休日の割増率を適用する。

```text
休日割増 = 概算時間単価 × 休日実働時間 × 休日割増率
```

通常勤務の場合、休日割増は0とする。

### 10.6 割増賃金合計

```text
割増賃金合計 = 深夜割増 + 時間外割増 + 休日割増
```

---

## 11. 控除計算

控除は利用者が任意で月額入力する。

対象項目：

- 健康保険
- 厚生年金
- 所得税
- 住民税
- その他控除

```text
控除合計 = 健康保険 + 厚生年金 + 所得税 + 住民税 + その他控除
```

---

## 12. 概算給与計算

### 12.1 概算総支給

```text
概算総支給 = 基本賃金相当 + 手当合計 + 割増賃金合計
```

### 12.2 概算手取り

```text
概算手取り = 概算総支給 - 控除合計
```

概算手取りが0未満になる場合は、0として表示するか、マイナス表示を許可するかを設定で選択可能とする。

Version 1.0ではマイナス表示を許可する。

## 15. 注意事項

本アプリの計算結果は概算であり、正式な給与額を保証するものではない。

以下の要素により、会社の給与明細と差異が発生する可能性がある。

- 会社側の端数処理
- 割増賃金の正式な計算方法
- 社会保険料の標準報酬月額
- 所得税の源泉徴収計算
- 住民税額
- 欠勤・遅刻・早退控除
- 有給休暇
- 特別手当
- 事故・違反判定
- 営業所固有の給与規程

本アプリは、乗務員が自分の給与見込みを把握するための補助ツールとして利用する。
