# 管理者向けマニュアル

## 1. 管理者設定を開く
設定画面の「管理者設定を表示」を押し、管理者コードを入力します。
初期コードは `0000` です。

## 2. 管理者設定で変更できる項目
- 運賃改定係数
- 給与算定係数
- 歩合給A率
- 歩合給B率
- B開始額
- B最高算定額
- 模範勤務手当/月
- 無事故手当/乗務
- 無違反手当/乗務
- 法定超 時間外割増
- 所定超 時間外割増
- 月60時間超 割増
- 法定休日割増
- 法定外休日割増
- 深夜割増

## 3. 変更時の注意
労働条件通知書、賃金規程、労使協定等に変更があった場合のみ変更してください。
通常利用者には管理者設定を操作させない運用を推奨します。

## 4. 公開時の注意
このアプリは端末内保存方式です。利用者の給与データはサーバーには保存されません。
公開URLを変更すると、利用者側では別アプリとして扱われる場合があります。


## 給与締め日の設定
Ver.1.1では締め日を以下の固定ルールで自動判定します。

- 原則：毎月15日
- 2月：14日
- 3月：16日

この項目は通常の管理者画面には表示せず、プログラム側の固定ルールとして保持します。
