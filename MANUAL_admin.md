# 開発者向け管理マニュアル Version 1.1

## 開発者設定を開く
利用者設定画面の「開発者設定」を押し、開発者パスワードを入力します。

## 管理項目
- 運賃改定係数
- 給与算定係数
- 消費税率
- 模範勤務手当、無事故手当、無違反手当
- 法定超、所定超、月60時間超、法定休日、法定外休日、深夜の各割増率

## 所得税表
`tax-table-2026.js` に国税庁令和8年分月額表を保持しています。年度更新時は、国税庁の最新「給与所得の源泉徴収税額表（月額表）」を確認し、表データ・高額給与の計算式・年度表示を一括更新してください。

参照元：国税庁「令和8年分 源泉徴収税額表」
https://www.nta.go.jp/publication/pamph/gensen/zeigakuhyo2026/01.htm

## GitHub Pages更新
更新版のファイル一式をリポジトリのルートへ上書きし、Commitします。Service Workerのキャッシュ名も変更すると、利用者端末へ更新が反映されやすくなります。

## 重要な制約
この版は静的PWAです。開発者パスワードはソースコードから完全に秘匿できません。ユーザー管理・課金・遠隔利用停止には、サーバー認証とバックエンドが必要です。
