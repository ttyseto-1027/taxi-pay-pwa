# 利用者マニュアル Version 1.1

## 初回設定
初回起動時に12種類から勤務区分を選択します。設定は端末内に保存されます。

## 毎日の入力
1. 勤務日を選択します。
2. 税込営収を入力します。税抜営収は自動表示されます。
3. 出勤時刻（アルコール）と退勤時刻（アルコール）を入力します。
4. 通常休憩と深夜休憩を、それぞれ時間・分で入力します。
5. 休日出勤の場合は「法定休日」または「法定外休日」を選択します。
6. 事故・違反があった乗務のみチェックします。
7. 「保存」をタップします。

## 設定
右上の⚙️から、健康保険、厚生年金、雇用保険、住民税、組合費、共済費、扶養親族等の数、源泉徴収区分、有休日数、標準報酬日額を設定できます。

所得税は国税庁の令和8年分月額表により自動計算されます。扶養控除等申告書を会社へ提出している場合は「甲欄」を選択してください。

## 結果確認
給与サマリーと「給与明細形式の内訳」に、勤務区分別歩合給、手当、割増賃金、所得税、控除、概算手取りが表示されます。

## 注意
本アプリは概算シミュレーションです。1か月単位の変形労働時間制や会社の勤務割、端数処理などにより、実際の給与明細と差が生じる場合があります。

## メイン画面の表示順
メイン画面は、上から「勤務実績を入力」「日別明細」「yyyy年mm月 給与シミュレーション」「給与締め履歴」の順で表示します。
