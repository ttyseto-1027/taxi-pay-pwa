# 利用者マニュアル（β版）

## 新規登録

1. アプリURLを開きます。
2. 「新規登録」を選びます。
3. 氏名、メールアドレス、本人専用のログインパスワードを入力します。
4. 管理者から渡されたβ版無料利用コードを入力します。
5. 利用条件とプライバシーポリシーへの同意欄を確認します。
6. 「登録して利用開始」を押します。

ログイン用パスワードとβ版無料利用コードは別のものです。

## ログイン

登録したメールアドレスと本人専用パスワードでログインします。端末上でログイン状態が維持される場合があります。

## 日々の入力

- 勤務日
- 税込営収
- 出勤時刻（アルコール）
- 退勤時刻（アルコール）
- 通常休憩
- 深夜休憩
- 休日区分
- 事故・違反の有無

給与・勤務データは原則として端末内に保存されます。

## 利用停止表示

「このアカウントは利用停止中です」と表示された場合は、アプリを利用できません。
