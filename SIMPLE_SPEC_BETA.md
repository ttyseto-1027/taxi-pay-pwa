# Taxi Payroll Processing Simulator
## ベータ版簡易仕様書 Version 1.0β

### 動作方式
- PWA
- Android / iPhoneのブラウザで利用
- オフライン利用対応
- 端末内保存

### 初回設定
12種類から勤務区分を選択し、端末内に保存する。

### 日報入力
- 勤務日
- 税込営収
- 出勤時刻（アルコール）
- 退勤時刻（アルコール）
- 通常休憩
- 深夜休憩
- 休日区分
- 事故・違反の有無

### 自動計算
- 税抜営収
- 月間営収
- 勤務区分別積算歩合給
- 諸手当
- 所定時間外、法定時間外、月60時間超、休日、深夜の割増賃金
- 所得税
- 概算総支給、控除合計、概算手取り

### 設定
- 社会保険料等
- 住民税、組合費、共済費、その他控除
- 扶養親族等の数、源泉徴収区分
- 有休日数、標準報酬日額

### ベータ版表示
個人名および問い合わせ先は表示しない。
