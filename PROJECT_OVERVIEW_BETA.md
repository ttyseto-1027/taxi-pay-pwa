# Taxi Payroll Processing Simulator
## ベータ版プロジェクト概要 Version 1.0β

### 1. 目的
タクシー乗務員が勤務実績と税込営収を入力し、給与支給額の概算を確認できるPWAを提供する。

### 2. 位置付け
本アプリは個人の給与確認を支援するシミュレーションツールであり、会社の給与計算システムや給与明細を代替しない。

### 3. 対象
隔日勤務、昼日勤、夜日勤および定時制9区分のタクシー乗務員。

### 4. ベータ版の運営方針
- サービス名義は Taxi Payroll Processing Simulator とする。
- 個人名は表示しない。
- 問い合わせ窓口は設けない。
- 利用者登録、課金、クラウド同期は実装しない。
- 入力データは原則として利用者端末内に保存する。

### 5. 主な機能
- 勤務区分12種類
- 税込営収入力、税抜営収自動計算
- 出勤・退勤時刻、通常休憩、深夜休憩
- 歩合給、諸手当、時間外・休日・深夜割増の概算
- 令和8年分源泉徴収税額表に基づく所得税概算
- 控除設定、有休加算
- 給与明細形式表示、CSV出力、印刷・PDF、給与締め履歴

### 6. 注意事項
計算結果は参考値であり、実際の給与支給額は会社が発行する給与明細で確認する。
