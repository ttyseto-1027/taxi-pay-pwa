# タクシー給与シミュレーター システム仕様書 Version 1.0

## 1. システム目的

本システムは、タクシー乗務員が個人のスマートフォンで給与概算を確認するためのPWAである。会社の正式な給与計算システムを代替するものではなく、乗務員個人の確認用ツールとして提供する。

## 2. システム構成

### 2.1 方式

- PWA（Progressive Web App）
- HTML / CSS / JavaScript
- 端末内ローカルストレージ保存
- オフライン利用対応
- サーバー側データベースなし
- ログイン機能なし

### 2.2 配布方式

- GitHub Pages等の静的ホスティングに配置
- 利用者はスマートフォンのブラウザでアクセス
- ホーム画面に追加してアプリ風に利用

### 2.3 対象端末

- iPhone Safari
- Android Chrome
- PCブラウザ（管理・確認用途）

## 3. 機能構成

### 3.1 利用者機能

- 給与サマリー表示
- 日報入力
- 日別明細表示
- 控除設定
- 給与締め
- 履歴表示
- 印刷 / PDF保存
- CSV出力

### 3.2 管理者機能

- 管理者設定の表示・編集
- 係数、率、手当額、割増率の管理
- 乗務員に見せる必要のない項目の非表示化

## 4. データ保存仕様

データはブラウザのlocalStorageにJSON形式で保存する。

保存対象は以下の通り。

- 利用者設定
- 管理者設定
- 日報データ
- 給与締め履歴

外部サーバーには保存しない。

## 5. オフライン仕様

Service Workerを使用し、以下のファイルをキャッシュする。

- index.html
- styles.css
- app.js
- manifest.json

初回アクセス後は、通信環境がない状態でも利用できることを目標とする。

## 6. 給与対象月判定

日報の日付から給与対象月を自動判定する。

- 原則、締め日は毎月15日
- 2月は14日
- 3月は16日

例：7月給与は6月16日から7月15日までを対象とする。

## 7. 管理者設定の考え方

利用者が混乱しないよう、係数や歩合率などは通常画面に表示しない。

通常画面には、日々入力する項目と結果表示を中心に配置する。

管理者設定には以下を含む。

- 消費税率
- 運賃改定係数
- 給与算定係数
- 歩合給A率
- 歩合給B率
- B開始額
- B最高算定額
- 模範勤務手当
- 無事故手当
- 無違反手当
- 各種割増率

## 8. セキュリティ・プライバシー

- 利用者データは本人端末にのみ保存
- サーバー送信なし
- 管理者が利用者データを見る機能なし
- ログイン情報なし
- 個人情報の収集なし

## 9. エラー処理

主な入力エラーは画面上で案内する。

- 日付未入力
- 税込営収未入力
- 時刻未入力
- 休憩時間が不正
- 帰庫時刻が翌日扱いになる場合の自動補正

## 10. 印刷・PDF仕様

ブラウザの印刷機能を利用する。

- 印刷ボタン押下でwindow.printを実行
- 利用者はブラウザ機能でPDF保存可能
- 印刷対象は給与サマリーおよび日別明細

## 11. CSV出力仕様

現在の給与対象月の日報データをCSV形式で出力する。

用途はバックアップ、Excel分析、給与明細照合補助とする。

## 12. 給与締め仕様

給与締め実行時、現在表示中の給与対象月の集計結果を履歴へ保存する。

日報データについては、運用方針により削除または保持を選択できる設計が望ましい。現行試作では履歴保存を優先し、設定値は保持する。
