# システム仕様書 Version 1.1

## 概要
タクシー乗務員が勤務実績と税込営収を入力し、端末内で給与概算を計算するPWA。ログインおよびサーバーデータベースは使用しない。

## 主な機能
- 初回勤務区分選択（通常3種、定時制9種）
- 日別勤務実績登録・編集・削除
- 税込から税抜への自動換算
- 勤務区分別積算歩合給
- 実働、深夜、時間外、休日労働の概算集計
- 国税庁令和8年分月額表による所得税自動計算
- 手当、控除、概算手取り
- 給与期間自動判定
- 給与締め履歴、CSV、印刷・PDF
- Service Workerによるオフライン利用

## 保存
LocalStorageへ利用者設定、開発者設定、勤務実績、給与締め履歴を保存する。入力データはサーバーへ送信しない。

## 制約
静的PWAの開発者パスワードは強固なサーバー認証ではない。割増賃金は会社の勤務割・法定休日指定を完全には再現せず概算とする。
